# Weather app

## Project Overview

This weather application allows users to search for a city and view its current weather conditions.
It also provides a five-day weather forecast and displays the top three most searched cities in a suggestion section.

## Setup Instructions

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

## Technical Stack

- TypeScript
- Vite
- React Router
- Sass
- Bootstrap
